对象

对象简介

对象将一组变量和函数组合起来，为你了解的真是世界中的某件事物创建模型。
在对象中：变量被认为是属性。属性向我们描述对象。
在对象中：函数被认为是方法 。方法代表和对象有关的任务。

创建对象

1.字面量语法
每个键和值之间用冒号分隔。
每个属性和方法之间用逗号分隔。

var hotel = {

    name: 'Quay',

    rooms: '40',

    booked: '25',

    checkAvailability: function(){

        return this.room-this.booked;

    }

};

2.构造函数语法
用new关键字和对象的构造函数相结合可以创建一个空白对象，随后可以为其添加属性和方法。

var hotel = new Object();

hotel.name = 'Quay';

hotel.rooms = 40;

hotel.booked = 25;

hotel.checkAvilability = function(){

    return this.rooms - this.booked;

};

可以用这种语法为你创建的对象添加属性和方法（无论是用哪种方法创建的）。

对象构造函数可以使用函数作为模板来创建对象。

function Hotel(name, rooms, booked){

    this.name = name;

    this.rooms = rooms;

    this.booked = booked;

    this.checkAvailability = function(){

        retuen this.rooms - this.booked;

    };

}

使用构造函数来创建对象的实例

var quayHotel = new Hotel('Quay', 40 , 25);

var parkHotel = new Hotel('Park', 120, 77);

访问对象

访问属性或方法时使用点符号。
也可以使用方括号来访问对象的属性（不能用来访问方法）。

var hotelName = hotel.name;

var roomsFree = hotel.checkAvailability();

var hotelName = hotel['name'];

修改对象

1.修改属性

hotel.name = 'Park';

如果这个属性不存在，则会将其添加到对象上。

2.删除属性

delete hotel.name;
